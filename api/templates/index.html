AI Image & Text API

Welcome to the AI Image Analysis & Generation API powered by Google Gemini.

This API provides endpoints for image analysis, text-to-image generation, image editing, and multi-image composition.

API Endpoints

GET /health

Check the health and capabilities of the API.

curl -X GET /health

GET /api

View complete API documentation.

curl -X GET /api

POST /ask

Answer text questions using Gemini AI.

Payload:

{
    "question": "Your question here",
    "model": "gemini-1.5-flash" (optional, default: gemini-1.5-flash)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "question": "What is the capital of France?" }' /ask

POST /analyze_base64

Analyze Base64 encoded images.

Payload:

{
    "image": "base64_encoded_image_string",
    "extract_masks": false (optional)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "image": "data:image/png;base64,iVBORw0KGgo...", "extract_masks": true }' /analyze_base64

POST /analyze_url

Analyze images from URLs.

Payload:

{
    "url": "image_url_string",
    "extract_masks": false (optional)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "url": "https://example.com/image.jpg" }' /analyze_url

POST /generate_text_to_image

Generate images from text prompts.

Payload:

{
    "prompt": "text_description",
    "style": "photorealistic" (optional),
    "aspect_ratio": "1:1" (optional)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "prompt": "A cat sitting on a moon", "style": "cartoon" }' /generate_text_to_image

POST /edit_image

Edit images using text prompts.

Payload:

{
    "prompt": "edit_description",
    "image": "base64_encoded_image",
    "style": "photorealistic" (optional),
    "aspect_ratio": "1:1" (optional),
    "edit_strength": 0.7 (optional)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "prompt": "Make the cat wear a hat", "image": "base64_image_data" }' /edit_image

POST /compose_images

Compose multiple images into one based on text prompt.

Payload:

{
    "prompt": "composition_description",
    "images": ["base64_image_1", "base64_image_2"],
    "style": "photorealistic" (optional),
    "aspect_ratio": "1:1" (optional)
}

Example:

curl -X POST -H "Content-Type: application/json" -d '{ "prompt": "Combine these into a futuristic city", "images": ["base64_image_1", "base64_image_2"] }' /compose_images

For more details, refer to the /api endpoint.
